# Muskick

##About

Muskick, a music search engine with a kick.

Week 8 project demonstrating usage of third-party APIs and libraries. Works best with a webkit enabled browser.

##APIs used

* Firebase
* BandsInTown
* OAuth.io
* Spotify
* Doorbell


##Libraries/frameworks used

* Bootstrap
* Chart.js
* Firebase.js
* JQuery
* JQueryUI
* Moment.js
